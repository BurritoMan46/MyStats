# MyStats

**_ScoreboardStats plugin for PocketMine_**

### MyStats supports:
 - MCPE 1.1 - 1.2
 - Api 3.0.0-ALPHA7, 3.0.0-ALPHA8
 - PocketMine, BlueLight
